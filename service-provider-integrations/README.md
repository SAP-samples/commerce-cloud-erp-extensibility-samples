# Service Provider Integration Samples

This folder contains sample plugins that demonstrate integration with external service providers, such as tax and data quality microservices. Each subfolder contains a sample plugin with its configuration and documentation.

## Included Samples

- `avalara-tax-sample-plugin/`: Sample plugin for Avalara tax integration.
- `dqm-address-cleansing-sample-plugin/`: Sample plugin for Data Quality Microservices integration.
